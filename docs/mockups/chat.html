你 · 14:30
运行 npm run db:migrate 时报错：

Error: P3009 migrate found failed migrations

该怎么办呀？🥺
🐱
猫猫助手 · DeepSeek · 14:30

喵～先别慌！ P3009 表示有失败的迁移记录，需要先处理它再 migrate 哦 🐾

推荐步骤：

1️⃣ 看看是哪条迁移失败了：

cd server
npx prisma migrate status

2️⃣ 修复 SQL 后标记回滚再重跑：

npx prisma migrate resolve --rolled-back "20250618000000_xxx"
npm run db:migrate

3️⃣ 开发环境可以直接重置（会清空数据！）：

npx prisma migrate reset
